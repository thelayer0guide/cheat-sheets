📋
RFC 4180 Format
Grundregeln
# Erste Zeile = optional Header name,alter,stadt "Müller, Hans",42,Berlin "Zeile mit Umbruch",30,Hamburg # Anführungszeichen escapen "Er sagte ""Hallo""",25,München
Trennzeichen-Varianten
CSV Komma (,) — Standard TSV Tabulator (\t) SSV Semikolon (;) — Excel DE PSV Pipe (|)
Encoding: UTF-8 ohne BOM ist Standard. Excel erzeugt oft UTF-8 mit BOM — kann Python-import stören.
🐍
Python — csv-Modul
Lesen
import csv with open('data.csv', newline='', encoding='utf-8') as f: reader = csv.DictReader(f) for row in reader: print(row['name'], row['alter']) # Semikolon-Trenner: reader = csv.DictReader(f, delimiter=';')
Schreiben
with open('out.csv', 'w', newline='', encoding='utf-8') as f: writer = csv.DictWriter(f, fieldnames=['name', 'alter']) writer.writeheader() writer.writerow({'name': 'Hans', 'alter': 42}) writer.writerows(liste_von_dicts)
pandas (komfortabel)
import pandas as pd df = pd.read_csv('data.csv', sep=';', encoding='utf-8') df.to_csv('out.csv', index=False) df['alter'].mean() df[df['alter'] > 30] df.groupby('stadt').size()
🔧
csvkit — CLI-Werkzeuge
Analyse
csvstat data.csv # Statistiken csvlook data.csv # Tabelle anzeigen csvstat --csv data.csv # als CSV ausgeben
Filtern & Auswählen
csvcut -c name,alter data.csv # Spalten auswählen csvcut -C stadt data.csv # Spalte ausschließen csvgrep -c stadt -m Berlin data.csv # Zeilen filtern csvgrep -c alter -r "^[3-9]" # Regex-Filter
Konvertieren
in2csv data.xlsx > out.csv # Excel → CSV in2csv --sheet Tabelle1 data.xlsx csvjson data.csv > data.json # CSV → JSON csvsort -c alter -r data.csv # sortieren (rev.) csvjoin -c id a.csv b.csv # JOIN über Spalte
SQL auf CSV
csvsql --query "SELECT name, COUNT(*) FROM data GROUP BY name" data.csv
⚡
Miller (mlr) — Transformation
Grundbefehle
mlr --csv head -n 5 data.csv mlr --csv sort -f name data.csv mlr --csv sort -nr alter data.csv # numerisch rev. mlr --csv cut -f name,alter data.csv mlr --csv filter '$alter > 30' data.csv
Transformieren
mlr --csv put '$voll = $name . " " . $stadt' data.csv mlr --csv rename alter,age data.csv mlr --csv stats1 -a mean,sum -f alter -g stadt data.csv mlr --icsv --ojson cat data.csv # → JSON
⚠️
Häufige Fallstricke
Encoding & Zeilenenden
BOM UTF-8 BOM: \xEF\xBB\xBF am Anfang CRLF Windows: \r\n statt \n encoding= latin-1 / cp1252 bei älteren Dateien # BOM entfernen: sed 's/\xEF\xBB\xBF//' input.csv > clean.csv # CRLF → LF: sed -i 's/\r//' data.csv
Quoting-Probleme
# Feld mit Komma MUSS gequotet sein: "Müller, Hans",42 # Anführungszeichen im Wert → doppeln: "Er sagte ""Hallo""" # Zeilenumbruch in Feld erlaubt (RFC 4180)
Wichtig: newline='' im Python-open() ist Pflicht beim csv-Modul, sonst entstehen Leerzeilen unter Windows.
