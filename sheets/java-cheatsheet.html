Streams & Lambdas
Lambda
Runnable r = () -> System.out.println("Hi"); Comparator<String> c = (a,b) -> a.compareTo(b);
Stream API
list.stream() .filter(s -> s.startsWith("A")) .map(String::toUpperCase) .sorted() .collect(Collectors.toList());
Aggregation
list.stream().count() list.stream().anyMatch(s -> s.isEmpty()) list.stream().reduce("", String::concat)
Optional
Optional<String> o = list.stream().findFirst(); o.ifPresent(System.out::println); o.orElse("default");

Kontrollstrukturen
for-each
for (String s : list) { ... } for (var entry : map.entrySet()) { entry.getKey(); entry.getValue(); }
switch expression (Java 14+)
int result = switch (day) { case MON, TUE -> 1; case WED -> 2; default -> 0; };

Exceptions
try-catch-finally
try { int r = 10 / 0; } catch (ArithmeticException e) { System.err.println(e.getMessage()); } finally { // immer ausgeführt }
try-with-resources
try (var br = new BufferedReader( new FileReader("f.txt"))) { br.lines().forEach(System.out::println); }
Eigene Exception
class AppException extends RuntimeException { public AppException(String msg) { super(msg); } }
