Sessions — Verwaltung
Neue Session starten
tmux tmux new # Kurzform tmux new-session tmux new -s mysession # mit Name
Session wiederherstellen (attach)
tmux attach tmux attach -t mysession tmux a -t mysession # Kurzform
Sessions auflisten
tmux ls tmux list-sessions
Session umbenennen / beenden
tmux rename-session -t old new tmux kill-session -t mysession tmux kill-server # ALLE beenden
Session-Shortcuts (im Prefix)
prefix d detach (Session verlassen) prefix $ Session umbenennen prefix s Sessions wechseln (Liste) prefix ( Vorherige Session prefix ) Nächste Session prefix L Letzte Session

⚙️
Scripting & CLI-Befehle
Session mit Befehl starten
tmux new -s dev -d # detached tmux new -s dev -d 'vim .'
Befehl in Session senden
tmux send-keys -t dev 'ls -la' Enter tmux send-keys -t dev:1.0 'cmd' Enter # session:window.pane
Fenster & Panes per Skript
tmux new-window -t dev -n logs tmux split-window -h -t dev:logs tmux select-window -t dev:logs tmux select-pane -t dev:logs.1
Infos abfragen
tmux list-sessions tmux list-windows -t mysession tmux list-panes -t mysession:0 tmux display-message -p '#S:#W.#P'
Pane-Inhalt speichern
tmux capture-pane -p > output.txt tmux capture-pane -pt dev:0.0
📝
~/.tmux.conf — Konfiguration
Prefix-Taste ändern
unbind C-b set -g prefix C-a bind C-a send-prefix
vi-Modus & Maus aktivieren
setw -g mode-keys vi set -g mouse on
Intuitive Split-Shortcuts
bind | split-window -h bind - split-window -v unbind '"' unbind %
Config neu laden
bind r source-file ~/.tmux.conf \; display-message "Config reloaded!"
Indexierung & History
set -g base-index 1 setw -g pane-base-index 1 set -g history-limit 50000
Statusbar
set -g status-position bottom set -g status-interval 5 set -g status-right '%H:%M %d-%b-%y'
True Color aktivieren
set -g default-terminal "tmux-256color" set -ag terminal-overrides ",xterm-256color:RGB"
💡 Config anwenden: prefix : source-file ~/.tmux.conf

TPM — Plugin Manager
TPM installieren
git clone https://github.com/tmux-plugins/tpm \ ~/.tmux/plugins/tpm
Plugins in ~/.tmux.conf definieren
set -g @plugin 'tmux-plugins/tpm' set -g @plugin 'tmux-plugins/tmux-sensible' set -g @plugin 'tmux-plugins/tmux-resurrect' set -g @plugin 'tmux-plugins/tmux-continuum' run '~/.tmux/plugins/tpm/tpm' # LETZTE ZEILE
Plugin-Shortcuts
prefix I Plugins installieren prefix U Plugins updaten prefix Alt+u Plugins entfernen
Beliebte Plugins
tmux-resurrect Sessions speichern/laden tmux-continuum Auto-Save + Auto-Restore tmux-yank Clipboard-Integration tmux-sensible Sinnvolle Defaults catppuccin Modernes Theme

Beispiel — Dev-Session per Skript
Vollständiges Setup-Skript
#!/usr/bin/env bash SESSION="dev" # Session erstellen (falls nicht vorhanden) tmux has-session -t $SESSION 2>/dev/null if [ $? != 0 ]; then tmux new-session -s $SESSION -d -n editor tmux send-keys -t $SESSION 'vim .' Enter tmux new-window -t $SESSION -n server tmux send-keys -t $SESSION 'npm run dev' Enter tmux new-window -t $SESSION -n shell tmux split-window -h -t $SESSION tmux select-window -t $SESSION:editor fi tmux attach -t $SESSION
💾
Sessions speichern & wiederherstellen
tmux-resurrect (Plugin erforderlich)
prefix Ctrl+s Session speichern prefix Ctrl+r Session wiederherstellen
tmux-continuum (Auto-Backup)
set -g @continuum-restore 'on' set -g @continuum-save-interval '15' # Speichert automatisch alle 15 Minuten
Manuell ohne Plugin (Workaround)
tmux list-sessions -F "#{session_name}" tmux display-message -p "#S" # Akt. Session
💡 Tipp: tmux-resurrect speichert auch vim-Sessions!
