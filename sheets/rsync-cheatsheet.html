⚡
Grundsyntax
Lokal → Lokal
rsync -av /src/ /dst/ # Inhalt von src rsync -av /src /dst/ # src-Ordner selbst
Lokal → Remote (SSH)
rsync -avz /local/ user@host:/remote/ rsync -avz -e "ssh -p 2222" /src/ host:/dst/
Remote → Lokal
rsync -avz user@host:/remote/ /local/
Trailing Slash Regel
# MIT Slash: Inhalt des Ordners rsync -av /src/ /dst/ → /dst/file1 /dst/file2 # OHNE Slash: den Ordner selbst rsync -av /src /dst/ → /dst/src/file1
🔧
Wichtige Optionen
Häufige Flags
-a archive (= -rlptgoD) -v verbose -z Komprimierung (SSH) -P Fortschritt + Partial (= --progress --partial) -n --dry-run (nur simulieren!) -u --update (überspringt neuere Zieldateien) -c --checksum (statt Mtime/Größe) -H Hardlinks beibehalten -A ACLs beibehalten -X Extended Attributes
Archiv = -rlptgoD
-r rekursiv -l Symlinks beibehalten -p Berechtigungen -t Timestamps -g Gruppe -o Owner (root) -D Gerätedateien + Spezial
Löschen
--delete # Zieldateien löschen, die Quelle fehlen --delete-before # vor dem Transfer löschen --delete-excluded # auch Excludes löschen
🔍
Include & Exclude
Inline Patterns
rsync -av --exclude='*.log' /src/ /dst/ rsync -av --exclude='.git/' --exclude='node_modules/' /src/ /dst/ rsync -av --include='*.py' --exclude='*' /src/ /dst/
Exclude-Datei
# .rsyncignore: *.log *.tmp .git/ node_modules/ __pycache__/ rsync -av --exclude-from='.rsyncignore' /src/ /dst/
Reihenfolge beachten
# Erst include, dann exclude! rsync -av \ --include='*.conf' \ --include='*/' \ --exclude='*' \ /etc/ /backup/etc/
💾
Backup-Strategien
Inkrementelles Backup (Hard-Link)
DATE=$(date +%Y-%m-%d) rsync -av --link-dest=/backup/latest \ /data/ /backup/$DATE/ ln -sfn /backup/$DATE /backup/latest
Mirror mit Löschung
rsync -avz --delete \ user@prod:/var/www/ \ /backup/www/
Bandbreite begrenzen
rsync -avz --bwlimit=5000 \ /src/ user@host:/dst/ # 5 MB/s
SSH-Optionen
rsync -avz \ -e "ssh -i ~/.ssh/backup_key -o StrictHostKeyChecking=no" \ /src/ backup@host:/dst/
Immer zuerst mit -n (dry-run) testen, besonders bei --delete!
