Basis-Typen

Interfaces & Type Aliases

type vs interface

Generics
Generische Funktion
function identity<T>(arg: T): T { return arg; } const r = identity<string>("hello");
Constraints
function getLen<T extends { length: number }>(t: T) { return t.length; }
keyof & typeof
type Keys = keyof User; // "id" | "name" | ... function get<T, K extends keyof T>(obj: T, key: K): T[K] { return obj[key]; }

Utility Types
Häufige Utilities

Conditional Types
type NonNull<T> = T extends null | undefined ? never : T;

Klassen
Class mit Access Modifiers

Abstract Class
abstract class Shape { abstract area(): number; toString() { return `area=${this.area()}`; } }

function isUser(x: unknown): x is User { return typeof x === "object" && x !== null && "id" in x; }
