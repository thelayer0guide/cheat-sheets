JSON-Syntax
Datentypen
{ "string": "Hallo Welt", "zahl": 42, "float": 3.14, "bool": true, "nichts": null, "array": [1, 2, 3], "object": { "a": 1 } }
Regeln
// Schlüssel: immer in doppelten "Anführungszeichen" // Kein trailing Komma erlaubt! // Keine Kommentare in JSON // Strings: nur " nie ' // Escape-Sequenzen: \n \t \\ \" \uXXXX
Formatieren / Validieren
jq '.' data.json # Pretty-print jq -c '.' data.json # Compact (eine Zeile) python3 -m json.tool data.json # Validieren + Format echo '{"a":1}' | jq # aus stdin
🔩
jq — Grundfilter
Zugriff
jq '.' # Identität (ganzes JSON) jq '.name' # Feld "name" jq '.a.b.c' # Tief verschachtelt jq '.arr[0]' # Erstes Array-Element jq '.arr[-1]' # Letztes Element jq '.arr[2:5]' # Slice jq '.arr[]' # Alle Elemente iterieren
Filtern & Auswählen
jq '.[] | select(.age > 30)' jq '.[] | select(.name | test("Mü"))' jq '.[].name' # Spalte "name" jq '.[] | {n:.name, a:.age}' # Projektion
Transformieren
jq 'map(.age * 2)' # auf Array anwenden jq 'map(select(.x > 0))' # filtern (Array out) jq '[.[] | .name]' # Namen als Array jq 'length' # Länge (Array/String) jq 'keys' # Schlüssel eines Objekts jq 'has("name")' # Feld vorhanden?
Aggregation
jq '[.[].age] | add' # Summe jq '[.[].age] | min, max' jq 'group_by(.city)' jq 'unique_by(.name)' jq '. | to_entries | map(select(.value != null))'
🐍
Python — json-Modul
Lesen & Schreiben
import json # Datei lesen with open('data.json', encoding='utf-8') as f: data = json.load(f) # Datei schreiben with open('out.json', 'w', encoding='utf-8') as f: json.dump(data, f, indent=2, ensure_ascii=False)
String-Konvertierung
# String → Python-Objekt obj = json.loads('{"name": "Hans"}') # Python-Objekt → String s = json.dumps(obj, indent=2, sort_keys=True) # Spezialtypen (datetime etc.) → str-Konverter: json.dumps(obj, default=str)

JSON Schema
Schema-Grundstruktur
{ "$schema": "https://json-schema.org/draft/2020-12/schema", "type": "object", "required": ["name", "alter"], "properties": { "name": { "type": "string", "minLength": 1 }, "alter": { "type": "integer", "minimum": 0 }, "email": { "type": "string", "format": "email" } }, "additionalProperties": false }
Typen & Keywords
type: string | number | integer | boolean | null | array | object enum: ["a", "b", "c"] pattern: "^[A-Z]{2}$" items: { "type": "string" } # Array-Einträge minItems / maxItems # Array-Länge anyOf / oneOf / allOf / not
Validieren (Python)
from jsonschema import validate, ValidationError try: validate(instance=data, schema=schema) except ValidationError as e: print(e.message)
