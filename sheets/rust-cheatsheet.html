Ownership & Borrowing
Ownership Regeln
let s1 = String::from("hello"); let s2 = s1; // s1 moved // println!("{}", s1); // Fehler!
Clone vs Copy
let s1 = String::from("hi"); let s2 = s1.clone(); // deep copy let i = 5; let j = i; // Copy (primitiv)
Referenzen & Borrows
fn len(s: &String) -> usize { s.len() } fn push(s: &mut String) { s.push('!'); } let mut s = String::from("hi"); push(&mut s);
Regel: Entweder EINE &mut Referenz ODER beliebig viele & Referenzen — nie beides gleichzeitig.

Typen & Variablen
Primitive
