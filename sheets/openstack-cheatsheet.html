Authentifizierung & Umgebung
RC-Datei einbinden
source openrc.sh
Umgebungsvariablen setzen
export OS_AUTH_URL=https://keystone:5000/v3 export OS_PROJECT_NAME=myproject export OS_USERNAME=admin export OS_PASSWORD=secret
Token anzeigen
openstack token issue
Katalog anzeigen
openstack catalog list
Aktives Projekt anzeigen
openstack configuration show
⚡ Kurzform mit Flags
openstack --os-cloud mycloud server list
⚡
Compute — Server (Nova)
Alle Server auflisten
openstack server list openstack server list --all-projects
Server erstellen
openstack server create \ --flavor m1.small \ --image ubuntu-22.04 \ --network mynet \ --key-name mykey \ my-server
Server-Details anzeigen
openstack server show my-server
Server starten / stoppen / neu starten
openstack server start my-server openstack server stop my-server openstack server reboot my-server openstack server reboot --hard my-server
Server löschen
openstack server delete my-server
Konsole öffnen
openstack console url show my-server openstack console log show my-server
Server resize (Flavor ändern)
openstack server resize \ --flavor m1.large my-server openstack server resize confirm my-server
Snapshot erstellen
openstack server image create \ --name my-snapshot my-server
SSH-Key hinzufügen
openstack keypair create \ --public-key ~/.ssh/id_rsa.pub mykey
🖼
Images (Glance) & Flavors
Images auflisten
openstack image list openstack image list --public
Image hochladen
openstack image create \ --file ubuntu.qcow2 \ --disk-format qcow2 \ --container-format bare \ --public ubuntu-22.04
Image anzeigen / löschen
openstack image show ubuntu-22.04 openstack image delete ubuntu-22.04
Flavors auflisten / erstellen
openstack flavor list openstack flavor create \ --ram 2048 --vcpus 2 \ --disk 20 m1.small
🌐
Netzwerk (Neutron)
Netzwerke auflisten
openstack network list openstack subnet list
Netzwerk erstellen
openstack network create mynet openstack subnet create \ --network mynet \ --subnet-range 192.168.1.0/24 \ mysubnet
Router erstellen & verbinden
openstack router create myrouter openstack router add subnet myrouter mysubnet openstack router set \ --external-gateway public myrouter
Floating IP verwalten
openstack floating ip create public openstack floating ip list openstack server add floating ip \ my-server 203.0.113.10
Security Groups
openstack security group list openstack security group create web-sg openstack security group rule create \ --proto tcp --dst-port 80 web-sg openstack server add security group \ my-server web-sg
Ports anzeigen
openstack port list --server my-server openstack port show PORT_ID
💾
Block Storage — Volumes (Cinder)
Volumes auflisten / erstellen
openstack volume list openstack volume create \ --size 50 my-volume
Volume anhängen / trennen
openstack server add volume \ my-server my-volume openstack server remove volume \ my-server my-volume
Volume-Snapshot
openstack volume snapshot create \ --volume my-volume my-snap openstack volume snapshot list
Volume-Typ & löschen
openstack volume type list openstack volume delete my-volume
👤
Identity — Users & Projekte (Keystone)
Benutzer verwalten
openstack user list openstack user create \ --password secret newuser openstack user show newuser openstack user delete newuser
Projekte verwalten
openstack project list openstack project create myproject openstack project delete myproject
Rollen zuweisen
openstack role list openstack role add \ --user newuser \ --project myproject \ member openstack role assignment list
Domain verwalten
openstack domain list openstack domain create mydomain
📦
Object Storage (Swift)
Container verwalten
openstack container list openstack container create mybucket openstack container show mybucket
Objekte hochladen / herunterladen
openstack object create mybucket file.txt openstack object list mybucket openstack object save \ mybucket file.txt openstack object delete mybucket file.txt

Quotas, Keypairs & Availability Zones
Quota anzeigen / setzen
openstack quota show openstack quota set \ --instances 50 \ --cores 100 myproject
Keypairs verwalten
openstack keypair list openstack keypair show mykey openstack keypair delete mykey
Availability Zones & Hosts
openstack availability zone list openstack host list openstack hypervisor list

Orchestration (Heat / Stacks)
Stack erstellen
openstack stack create \ -t template.yaml my-stack
Stacks verwalten
openstack stack list openstack stack show my-stack openstack stack update \ -t new-template.yaml my-stack openstack stack delete my-stack
Stack Events & Ressourcen
openstack stack event list my-stack openstack stack resource list my-stack
⚖️
Load Balancer (Octavia)
Load Balancer erstellen
openstack loadbalancer create \ --name my-lb \ --vip-subnet-id SUBNET_ID
Listener & Pool
openstack loadbalancer listener create \ --protocol HTTP --protocol-port 80 \ --name my-listener my-lb openstack loadbalancer pool create \ --lb-algorithm ROUND_ROBIN \ --listener my-listener \ --protocol HTTP
Member hinzufügen
openstack loadbalancer member create \ --address 192.168.1.10 \ --protocol-port 8080 POOL_ID
🎛
Output-Formate & Filterung
Format wählen
openstack server list -f table # default openstack server list -f json openstack server list -f yaml openstack server list -f csv openstack server list -f value
Spalten auswählen
openstack server list \ -c ID -c Name -c Status
Filtern
openstack server list \ --status ACTIVE openstack server list \ --name my-server
Nur Wert ausgeben (für Skripte)
openstack server show my-server \ -f value -c id
💡 Tipp: Mit --debug werden alle API-Calls ausgegeben
🔤
DNS (Designate)
Zonen verwalten
openstack zone list openstack zone create \ --email admin@example.com \ example.com.
DNS-Einträge
openstack recordset list example.com. openstack recordset create \ --type A --record 1.2.3.4 \ example.com. www

Nützliche Kombinationen
Server-IDs aller aktiven Server
openstack server list \ --status ACTIVE \ -f value -c ID

FIP=$(openstack floating ip create \ public -f value -c floating_ip_address) openstack server add floating ip \ my-server $FIP
Alle Ressourcen eines Projekts
openstack server list --project myproject openstack volume list --project myproject openstack floating ip list \ --project myproject
