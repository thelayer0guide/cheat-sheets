Setup & Auth
Installation
brew install govc # oder binary download: curl -L https://github.com/vmware/govmomi/releases/latest/download/govc_Linux_x86_64.tar.gz | tar xz mv govc /usr/local/bin/
Umgebungsvariablen
export GOVC_URL=https://vcenter.example.com export GOVC_USERNAME=administrator@vsphere.local export GOVC_PASSWORD=secret export GOVC_INSECURE=true # self-signed cert export GOVC_DATACENTER=DC01 export GOVC_DATASTORE=ds01 export GOVC_NETWORK="VM Network" export GOVC_RESOURCE_POOL=/DC01/host/cluster01/Resources
Verbindung testen
govc about govc about -u https://user:pass@vcenter/sdk
Tipp: GOVC_URL, USERNAME und PASSWORD in eine .env-Datei auslagern und mit `source .env` laden. Nie Passwörter in die Shell-History.
🔍
Navigation & Suche
Inventar durchsuchen
govc ls # Datacenter-Ebene govc ls /DC01/vm govc ls /DC01/vm/folder01 govc ls /DC01/host govc ls /DC01/datastore govc ls /DC01/network
Objekte finden
govc find / -type m # alle VMs govc find / -type m -name myvm* govc find / -type h # Hosts govc find / -type d # Datastores govc find /DC01/vm -type m -runtime.powerState poweredOn govc find / -type m -guest.ipAddress 10.0.0.5
Object-Typen
m VirtualMachine h HostSystem d Datastore n Network r ResourcePool f Folder c ClusterComputeResource
🖥
VMs verwalten
Info & Status
govc vm.info /DC01/vm/myvm govc vm.info -json myvm | jq . govc vm.ip myvm
Power-Operationen
govc vm.power -on myvm govc vm.power -off myvm govc vm.power -s myvm # graceful shutdown govc vm.power -r myvm # reboot govc vm.power -suspend myvm
Klonen & Zerstören
govc vm.clone -vm template-vm -on=false newvm govc vm.clone -vm myvm \ -ds ds01 \ -pool /DC01/host/cluster01/Resources \ myvm-clone govc vm.destroy myvm
Konfigurieren
govc vm.change -vm myvm -m 8192 -c 4 govc vm.change -vm myvm -e disk.enableUUID=TRUE govc vm.disk.change -vm myvm -disk.label "Hard disk 1" -size 100G govc vm.network.add -vm myvm -net "PG-Prod" govc vm.migrate -host esx02 myvm

Datastores & Dateien
Datastore-Inhalt
govc datastore.ls govc datastore.ls -ds ds01 govc datastore.ls -ds ds01 myvm/ govc datastore.info ds01
Dateien transferieren
govc datastore.upload -ds ds01 local.iso /isos/ubuntu.iso govc datastore.download -ds ds01 /myvm/myvm.vmdk local.vmdk govc datastore.mkdir -ds ds01 newfolder govc datastore.rm -ds ds01 oldfile.iso

Snapshots
Snapshots verwalten
govc snapshot.tree -vm myvm govc snapshot.create -vm myvm pre-patch govc snapshot.revert -vm myvm pre-patch govc snapshot.remove -vm myvm pre-patch govc snapshot.remove -vm myvm -r "" # alle entfernen

Import & Export
OVA/OVF Import
govc import.ova ubuntu.ova govc import.ova \ -name myvm \ -ds ds01 \ -pool /DC01/host/cluster01/Resources \ ubuntu-22.04.ova # OVF mit Spec-Datei: govc import.spec ubuntu.ova > spec.json govc import.ova -options spec.json ubuntu.ova
OVF Export
govc export.ovf -vm myvm ./export-dir/
Content Library
govc library.ls govc library.info mylib govc library.import mylib /path/to/file.ova govc library.deploy mylib/template-name myvm

Host, Cluster & Tags
Hosts & Cluster
govc host.info /DC01/host/cluster01/esx01.example.com govc host.storage.info -host esx01 govc cluster.usage cluster01
Tags (vSphere Tags API)
govc tags.category.ls govc tags.ls -c Environment govc tags.attach -c Environment -t Production /DC01/vm/myvm govc tags.info /DC01/vm/myvm
Events & Tasks
