Skalare & Typen
Grundtypen
string: Hallo Welt # unquoted OK quoted: "Hallo\nWelt" # mit Escape literal: 'kein Escape: \n' # single quote integer: 42 float: 3.14 bool: true # YAML 1.1: yes | no | on | off auch bool null: null # ~ ist auch null datum: 2025-01-15 # ISO 8601
Explizite Typen (Tags)
als_string: !!str 42 # Zahl → String als_int: !!int "42" # String → Int als_float: !!float 1 # → 1.0 als_bool: !!bool "yes" # → true

Sequenzen & Mappings
Block-Stil (empfohlen)
# Mapping (Dictionary) name: Hans alter: 42 adresse: strasse: Hauptstr. 1 plz: 12345 # Sequenz (Liste) hobbys: - Lesen - Kochen - YAML schreiben
Flow-Stil (kompakt)
# Mapping inline person: {name: Hans, alter: 42} # Sequenz inline farben: [rot, grün, blau] # Gemischt teams: - {name: Alpha, size: 5} - {name: Beta, size: 3}
📝
Mehrzeilige Strings
Literal Block (|) — Zeilenenden erhalten
text: | Erste Zeile Zweite Zeile Dritte Zeile # → "Erste Zeile\nZweite Zeile\nDritte Zeile\n"
Folded Block (>) — Umbrüche zu Leerzeichen
text: > Diese lange Zeile wird zusammengefaltet. # → "Diese lange Zeile wird zusammengefaltet.\n"
Chomping-Modifier
clip: | # Standard: ein \n am Ende strip: |- # Kein \n am Ende keep: |+ # Alle \n am Ende erhalten

Anchor & Alias
# Anchor definieren defaults: &defaults timeout: 30 retries: 3 # Alias verwenden produktion: <<: *defaults # Merge Key host: prod.example.com staging: <<: *defaults host: staging.example.com timeout: 60 # überschreibt

Python — PyYAML / ruamel
PyYAML (einfach)
import yaml # Lesen with open('config.yaml') as f: cfg = yaml.safe_load(f) # Schreiben with open('out.yaml', 'w') as f: yaml.dump(data, f, allow_unicode=True, default_flow_style=False) # String → Objekt obj = yaml.safe_load("key: value")
Mehrere Dokumente
# Alle Dokumente laden docs = list(yaml.safe_load_all(f)) # Alle Dokumente schreiben yaml.dump_all([doc1, doc2], f)

Häufige Fallstricke
Das Norway-Problem
# YAML 1.1 interpretiert "NO" als false! land: NO # → false (YAML 1.1 Bug) land: "NO" # → "NO" (korrekt, gequotet) # Weitere Booleans in YAML 1.1: # yes, no, on, off, true, false (und Großbuchstaben) # YAML 1.2 (ruamel.yaml): nur true/false — PyYAML bleibt YAML 1.1
Einrückung & Tabulatoren
# NIEMALS Tabs zur Einrückung! # Nur Leerzeichen erlaubt # Einrücktiefe konsistent halten (2 oder 4) # Doppelpunkt im Wert → quoten: url: "https://example.com:8080" # Sonderzeichen am Anfang → quoten: wert: "{nicht als Objekt}"
Tipp: yaml.safe_load() statt yaml.load() verwenden — verhindert Code-Ausführung aus nicht-vertrauenswürdigen YAML-Dateien.
