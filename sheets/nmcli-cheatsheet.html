Allgemein & Status
Allgemeiner Status
nmcli general status nmcli g # Kurzform
Networking On/Off (Global)
nmcli networking on # Aktivieren nmcli networking off # Deaktivieren nmcli networking connectivity
Funknetzwerke (Radio)
nmcli radio nmcli radio wifi on # WLAN an nmcli radio wifi off # WLAN aus nmcli radio wwan on # WWAN an
Live-Monitor
nmcli monitor # Events loggen
🔗
Verbindungen (Connections)
Verbindungen anzeigen
nmcli connection show nmcli c # Alle nmcli c show --active # Nur aktive nmcli c show "Name" # Details zu "Name"
Verbindung herstellen / trennen
nmcli c up "Name" # Starten nmcli c down "Name" # Stoppen
Statische IP / IPv4 Konfigurieren
# Neue Ethernet Verbindung mit statischer IP definieren nmcli c add type ethernet con-name "static-eth0" ifname eth0 nmcli c mod "static-eth0" \ ipv4.addresses 192.168.1.100/24 \ ipv4.gateway 192.168.1.1 \ ipv4.dns "8.8.8.8 1.1.1.1" \ ipv4.method manual
DHCP Konfigurieren
nmcli c mod "Name" ipv4.method auto
Verbindung bearbeiten & löschen
nmcli c delete "Name" # Löschen nmcli c edit "Name" # Interaktive Shell
💡 nmcli c reload — Konfigurationen von Festplatte neu laden.
⚙️
Geräte (Devices)
Gerätestatus
nmcli device status nmcli d # Kurzform
Gerätedetails
nmcli d show # Alle Schnittstellen nmcli d show eth0 # Nur eth0
Verbinden / Trennen (Physisches Interface)
nmcli d connect eth0 # Startet Auto-Connect nmcli d disconnect eth0 # Trennt bestehende Verbindung
Gerät modifizieren
nmcli d modify eth0 ipv4.method auto nmcli d modify eth0 managed yes
🛜
WLAN (Wi-Fi)
Netzwerke Suchen
nmcli device wifi list nmcli d wifi list # Alias nmcli d wifi rescan # Scan erzwingen
Verbinden mit SSID
nmcli d wifi connect "SSID" password "geheim123" # ODER wenn bereits das Passwort gespeichert ist: nmcli c up "SSID"
Verstecktes Netzwerk
nmcli d wifi connect "SSID" password "geheim123" hidden yes
WLAN Hotspot erstellen
nmcli d wifi hotspot ifname wlan0 ssid "MeinHotspot" password "pass123"
🛠
Troubleshooting & Logs
Logs einsehen
journalctl -u NetworkManager journalctl -f -u NetworkManager # Live folgen
Kurze Ausgabe (Skripte)
nmcli -t -f NAME,TYPE,STATE c show # Tabular & fields
