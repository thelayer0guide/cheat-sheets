PromQL — Grundlagen
Einfache Abfragen
# Alle Zeitreihen einer Metrik: node_cpu_seconds_total # Mit Label-Filter: node_cpu_seconds_total{mode="idle"} node_cpu_seconds_total{job="node",mode!="idle"} # Regex-Filter: node_cpu_seconds_total{mode=~"user|system"} node_cpu_seconds_total{instance!~".*:9100"}
Range Vectors (Zeitfenster)
# Letzte 5 Minuten: http_requests_total[5m] # Rate über 5 Minuten: rate(http_requests_total[5m]) # Instant Rate (irate - letztes Intervall): irate(http_requests_total[5m])
Arithmetik & Vergleiche
# CPU-Auslastung in %: 100 - (avg by (instance) ( irate(node_cpu_seconds_total{mode="idle"}[5m]) ) * 100) # Schwellenwert-Vergleich: node_memory_MemAvailable_bytes < 500000000
🔢
Aggregationen
Aggregationsoperatoren
sum(http_requests_total) avg(node_memory_MemAvailable_bytes) max(node_load1) min(node_filesystem_free_bytes) count(up == 1) topk(5, http_requests_total)
Gruppierung (by / without)
sum by (instance) (node_cpu_seconds_total) sum by (job, instance) (up) avg without (cpu, mode) (node_cpu_seconds_total) # Rate pro Job: sum by (job) ( rate(http_requests_total[5m]) )
Quantilen
# P95 der Request-Dauer: histogram_quantile(0.95, rate(http_request_duration_seconds_bucket[5m]) ) histogram_quantile(0.99, sum by (le) ( rate(http_request_duration_seconds_bucket[5m]) ) )
🔧
Funktionen
Zeit & Änderungsrate
increase(http_requests_total[1h]) delta(node_temperature_celsius[10m]) deriv(node_memory_MemFree_bytes[5m]) predict_linear(node_filesystem_free_bytes[1h], 4*3600)
Labels & Metadaten
label_replace(up, "host", "$1", "instance", "(.*):.*") label_join(up, "fullname", "-", "job", "instance") absent(up{job="node"})
Mathematik
round(node_memory_MemAvailable_bytes / 1024^3, 0.01) clamp_max(node_cpu_seconds_total, 100) floor(rate(http_requests_total[5m]))
🚨
Alerting Rules
Alert-Regel (rules.yml)
groups: - name: node-alerts rules: - alert: HighCPU expr: | 100 - (avg by(instance)( irate(node_cpu_seconds_total{mode="idle"}[5m]) ) * 100) > 90 for: 5m labels: severity: critical annotations: summary: "CPU > 90% on {{ $labels.instance }}" description: "Value: {{ $value }}"
Alertmanager routing (alertmanager.yml)
route: group_by: ['alertname', 'instance'] group_wait: 30s receiver: 'slack' routes: - match: severity: critical receiver: 'pagerduty' receivers: - name: 'slack' slack_configs: - api_url: 'https://hooks.slack.com/...' channel: '#alerts'

Recording Rules
Recording Rule (für teure Queries)
groups: - name: cpu_recording interval: 1m rules: - record: job:node_cpu_usage:avg expr: | 100 - (avg by (job) ( irate(node_cpu_seconds_total{mode="idle"}[5m]) ) * 100) # Naming: level:metric:operations
