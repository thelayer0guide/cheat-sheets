Server auflisten & anzeigen
Übersicht
nova list nova list --all-tenants nova list --status ERROR nova list --host compute01.example.com nova list --name "myapp*" nova show myserver
Filter
nova list --image IMAGE-UUID nova list --flavor m1.large nova list --tenant PROJECT-UUID nova list --user USER-UUID nova list --fields name,status,networks
Admin: Hypervisors
nova hypervisor-list nova hypervisor-show compute01.example.com nova hypervisor-stats nova host-list nova host-describe compute01.example.com
🚀
Server booten (nova boot)
Minimal
nova boot \ --flavor m1.small \ --image ubuntu-22.04 \ --nic net-id=NET-UUID \ --key-name mykey \ myserver
Vollständig
nova boot \ --flavor m1.large \ --image ubuntu-22.04 \ --nic net-id=NET-UUID \ --security-groups default,websg \ --key-name mykey \ --availability-zone az1 \ --user-data ./cloud-init.yml \ --meta env=prod \ --meta team=ops \ --poll \ myserver
Boot from Volume
nova boot \ --flavor m1.small \ --block-device \ source=image,id=IMAGE-UUID,\ dest=volume,size=50,\ shutdown=remove,bootindex=0 \ --nic net-id=NET-UUID \ myserver-bfv
Aus bestehendem Volume
nova boot \ --flavor m1.small \ --block-device \ source=volume,id=VOL-UUID,\ dest=volume,bootindex=0 \ --nic net-id=NET-UUID \ myserver-vol
⚡
Power-Operationen
Start / Stop / Reboot
nova start myserver nova stop myserver nova reboot myserver # soft reboot nova reboot --hard myserver # hard reset
Pause / Suspend / Shelve
nova pause myserver # RAM eingefroren, CPU gestoppt nova unpause myserver nova suspend myserver # Zustand auf Disk gespeichert nova resume myserver nova shelve myserver # Snapshot + Hypervisor freigeben nova unshelve myserver nova shelve-offload myserver
Löschen
nova delete myserver nova force-delete myserver # erzwungen, kein ACTIVE-Check nova delete server1 server2 server3
Rebuild & Rename
nova rebuild myserver ubuntu-22.04 nova rebuild --poll myserver IMAGE-UUID nova rename myserver newname
🖥
Console & Logs
VNC / SPICE / Serial
nova get-vnc-console myserver novnc nova get-vnc-console myserver xvpvnc nova get-spice-console myserver spice-html5 nova get-serial-console myserver serial
Console-Log
nova console-log myserver nova console-log --length 50 myserver
Tipp: nova get-vnc-console gibt eine URL zurück, die im Browser geöffnet werden kann. novnc ist für HTML5-Browser, xvpvnc für Java-Clients.
📐
Resize & Migration
Resize (kalt)
nova resize myserver m1.large --poll nova resize-confirm myserver # nach erfolgreichem Resize nova resize-revert myserver # Resize rückgängig machen
Cold Migration
nova migrate myserver nova migrate --host compute02.example.com myserver
Live Migration
nova live-migration myserver nova live-migration myserver compute02.example.com nova live-migration --block-migrate myserver # ohne Shared Storage
Evacuate (Host ausgefallen)
nova evacuate myserver nova evacuate --on-shared-storage myserver nova evacuate myserver compute03.example.com

Flavors
Auflisten & Anzeigen
nova flavor-list nova flavor-list --all # inkl. privater Flavors nova flavor-show m1.large
Erstellen
# nova flavor-create NAME ID RAM_MB DISK_GB VCPUS nova flavor-create m1.xlarge auto 8192 80 4 nova flavor-create \ --ephemeral 20 --swap 1024 \ --is-public true \ m1.xlarge auto 8192 80 4
Extra-Specs / Zugriff
nova flavor-key m1.xlarge set hw:cpu_policy=dedicated nova flavor-key m1.xlarge set hw:mem_page_size=large nova flavor-key m1.xlarge unset hw:cpu_policy nova flavor-access-add m1.secret PROJECT-UUID nova flavor-access-remove m1.secret PROJECT-UUID nova flavor-access-list m1.secret
Löschen
nova flavor-delete m1.oldflav
