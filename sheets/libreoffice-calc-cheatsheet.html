Formeln — Grundlagen
Summe / Mittelwert / Anzahl
=SUM(B2:B100) =AVERAGE(B2:B100) =COUNT(B2:B100) # nur Zahlen =COUNTA(B2:B100) # alle nichtleeren =MAX(B2:B100) =MIN(B2:B100)
Wenn-Funktion
=IF(A2>100,"Hoch","Niedrig") =IF(A2="","leer",A2) =IFS(A2>90,"A",A2>70,"B",TRUE,"C")
Fehlerbehandlung
=IFERROR(A2/B2,0) =IFNA(VLOOKUP(…),"Nicht gefunden") =ISERROR(A2) # → WAHR/FALSCH
Absolute / relative Bezüge
=A2*B2 # relativ =$A$2*B2 # A2 absolut =$A2*B2 # Spalte absolut =A$2*B2 # Zeile absolut # F4 wechselt den Bezugstyp
Tipp: Mit Ctrl+` (Grave Accent) wechselt die Ansicht zwischen Formeln und Werten.

Lookup-Funktionen
VLOOKUP (senkrecht)
=VLOOKUP(Suchkrit,Matrix,SpaltenIdx,0) # 0 = exakte Übereinstimmung # 1 = ungefähr (sortiert!) =VLOOKUP(A2,$D$2:$F$100,2,0)
HLOOKUP (waagerecht)
=HLOOKUP(Suchkrit,Matrix,ZeilenIdx,0) # sucht in erster Zeile der Matrix
INDEX / MATCH
=INDEX(C2:C100,MATCH(A2,B2:B100,0)) # MATCH gibt Zeilennummer zurück # INDEX holt den Wert an der Position # MATCH-Typ: 0=exakt, 1=≤, -1=≥
XLOOKUP (ab LO 24.2)
=XLOOKUP(Suchkrit,SuchArr,RückgabeArr) =XLOOKUP(A2,D2:D100,E2:E100,"N/A") # 4. Arg: Wert wenn nicht gefunden # 5. Arg: 0=exakt, -1=exakt/kleiner
MATCH / XMATCH
=MATCH("Berlin",A2:A100,0) =XMATCH("Berlin",A2:A100,0) # gibt relative Position zurück

Statistische Funktionen
SUMIF / COUNTIF
=SUMIF(A2:A100,"Berlin",B2:B100) =COUNTIF(A2:A100,">100") =COUNTIF(A2:A100,"Ber*") # Wildcard
SUMIFS / COUNTIFS (mehrere Kriterien)
=SUMIFS(B2:B100,A2:A100,"DE",C2:C100,">0") =COUNTIFS(A2:A100,">=2024-01-01", A2:A100,"<=2024-12-31") # Summe-Bereich muss gleich groß sein
AVERAGEIF / AVERAGEIFS
=AVERAGEIF(A2:A100,">0",B2:B100) =AVERAGEIFS(B2:B100,A2:A100,"DE", C2:C100,">10")
Rang / Große / Kleine
=RANK(A2,A2:A100,0) # 0=absteigend =LARGE(B2:B100,3) # 3. größter Wert =SMALL(B2:B100,1) # kleinster Wert
Tipp: Kriterien in SUMIFS/COUNTIFS können Platzhalter enthalten: * (beliebig viele Zeichen), ? (ein Zeichen).

Datum & Zeit
Aktuelles Datum / Zeit
=TODAY() # heutiges Datum =NOW() # Datum + aktuelle Uhrzeit Ctrl+; # aktuelles Datum als fester Wert Ctrl+Shift+; # aktuelle Uhrzeit als fester Wert
Datum konstruieren / zerlegen
=DATE(2024,12,31) # → serielles Datum =YEAR(A2) # Jahr =MONTH(A2) # Monat (1–12) =DAY(A2) # Tag (1–31)
Differenz / Versatz
=DATEDIF(A2,B2,"D") # Tage =DATEDIF(A2,B2,"M") # Monate =DATEDIF(A2,B2,"Y") # Jahre =EDATE(A2,3) # +3 Monate =EOMONTH(A2,0) # Monatsende
Wochentag / Arbeitstage
=WEEKDAY(A2,2) # 2=Mo=1, So=7 =WORKDAY(A2,10) # +10 Arbeitstage =NETWORKDAYS(A2,B2) # Arbeitstage zw.
Hinweis: Datumsangaben in LibreOffice Calc sind intern Ganzzahlen (Tage seit 30.12.1899). Formatierung entscheidet über die Darstellung.

Text-Funktionen
Verbinden
=A2&" "&B2 # & ist Concat =CONCAT(A2," ",B2) =TEXTJOIN(";",TRUE,A2:A10) # Trennzeichen
Extrahieren
=LEFT(A2,5) # erste 5 Zeichen =RIGHT(A2,3) # letzte 3 Zeichen =MID(A2,3,4) # ab Pos 3, 4 Zeichen =LEN(A2) # Zeichenanzahl
Suchen / Ersetzen
=FIND("@",A2) # Pos (Groß-/Klein) =SEARCH("@",A2) # Pos (unabhängig) =SUBSTITUTE(A2,"alt","neu") =REPLACE(A2,3,2,"XY") # ab Pos 3, 2 Zeichen
Bereinigen / Umwandeln
=TRIM(A2) # Leerzeichen entfernen =CLEAN(A2) # Steuerzeichen entfernen =UPPER(A2) # GROSSBUCHSTABEN =LOWER(A2) # kleinbuchstaben =PROPER(A2) # Erster Gross
TEXT-Formatierung
=TEXT(A2,"TT.MM.JJJJ") # Zahl als Text =TEXT(A2,"0,00 €") # Währung als Text =VALUE("123,45") # Text → Zahl
