Variablen & Typen
Deklaration
const name = "Alice"; // unveränderlich let count = 0; // veränderlich // var vermeiden (function-scoped)
Typen
typeof "hello" // "string" typeof 42 // "number" typeof true // "boolean" typeof null // "object" (Bug!) typeof undefined // "undefined"
Destructuring
const [a, b, ...rest] = [1,2,3,4]; const { x, y = 10 } = point; const { name: n } = person; // rename
Template Literals
const msg = `Hallo ${name}, du bist ${age}!`; const html = ` <div>${content}</div> `;
⚙️
Funktionen
Arrow Functions
const add = (a, b) => a + b; const greet = name => `Hi ${name}`; const noop = () => {};
Default & Rest Parameter
function greet(name = "Welt") { ... } function sum(...nums) { return nums.reduce((a,b) => a+b, 0); }
Spread
const arr = [1, ...other, 2]; const obj = { ...base, x: 1 }; Math.max(...nums);
📦
Arrays
Methoden
arr.map(x => x * 2) arr.filter(x => x > 0) arr.reduce((acc,x) => acc+x, 0) arr.find(x => x.id === id) arr.some(x => x > 10) arr.every(x => x > 0) arr.includes(42) arr.flat() arr.flatMap(fn)
Sortieren & Suchen
arr.sort((a,b) => a - b); // numerisch arr.indexOf(42) arr.findIndex(x => x.id===id) arr.slice(1, 3) // kein Mutieren arr.splice(1, 1, "new") // Mutieren
🔑
Objects & Classes
Object Methoden
Object.keys(obj) Object.values(obj) Object.entries(obj) Object.assign({}, defaults, opts) Object.freeze(obj)
Klasse (ES6)
class Animal { constructor(name) { this.name = name; } speak() { return `${this.name}!`; } static create(n) { return new Animal(n); } } class Dog extends Animal { speak() { return `${super.speak()} Wuff`; } }
Optional Chaining & Nullish
user?.address?.city user?.getName?.() arr?.[0] val ?? "default" // nur bei null/undefined

Async / Await & Promises
async/await
async function fetchUser(id) { try { const res = await fetch(`/api/${id}`); return await res.json(); } catch (err) { console.error(err); } }
Promise kombinieren
Promise.all([p1, p2, p3]) Promise.allSettled([p1, p2]) Promise.race([p1, p2]) Promise.any([p1, p2])
fetch API
const data = await fetch(url, { method: "POST", headers: { "Content-Type": "application/json" }, body: JSON.stringify(payload) }).then(r => r.json());
