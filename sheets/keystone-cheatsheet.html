👤
Users
Auflisten & Anzeigen
keystone user-list keystone user-show alice keystone user-show --tenant myproject alice
Erstellen & Ändern
keystone user-create \ --name alice \ --pass mysecret \ --email alice@example.com \ --enabled true keystone user-update \ --pass newsecret \ --email new@example.com alice keystone user-update --enabled false alice keystone user-update --enabled true alice keystone user-delete alice
Passwort ändern
keystone user-password-update \ --pass newsecret alice
Hinweis: keystone CLI ist seit OpenStack Ocata deprecated. Für neue Installationen wird openstack CLI (python-openstackclient) empfohlen.
📁
Tenants (Projects)
Auflisten & Anzeigen
keystone tenant-list keystone tenant-show myproject
Erstellen & Ändern
keystone tenant-create \ --name myproject \ --description "My Project" \ --enabled true keystone tenant-update \ --description "Updated" myproject keystone tenant-update --enabled false myproject keystone tenant-delete myproject
🎭
Roles & Assignments
Rollen verwalten
keystone role-list keystone role-show member keystone role-create myrole keystone role-delete myrole
Rolle zuweisen
keystone user-role-add \ --user alice \ --role member \ --tenant myproject keystone user-role-remove \ --user alice \ --role member \ --tenant myproject
Zuweisungen anzeigen
keystone user-role-list keystone user-role-list \ --user alice \ --tenant myproject
🎫
Token & Catalog
Token
keystone token-get # Gibt Token-ID, Tenant, User, Expires aus # credentials.sh / openrc (v2): export OS_AUTH_URL=http://keystone:5000/v2.0 export OS_TENANT_NAME=myproject export OS_USERNAME=alice export OS_PASSWORD=mysecret
Service Catalog
keystone catalog keystone catalog --service compute keystone service-list
🌐
Endpoints & Services
Endpoints
keystone endpoint-list keystone endpoint-get --service compute keystone endpoint-create \ --service compute \ --region RegionOne \ --publicurl http://nova:8774/v2/%(tenant_id)s \ --internalurl http://nova-int:8774/v2/%(tenant_id)s \ --adminurl http://nova-admin:8774/v2/%(tenant_id)s keystone endpoint-delete ENDPOINT-UUID
Services
keystone service-list keystone service-show compute keystone service-create \ --name nova \ --type compute \ --description "Nova Compute" keystone service-delete SERVICE-UUID
EC2 Credentials
keystone ec2-credentials-create \ --tenant myproject keystone ec2-credentials-list keystone ec2-credentials-delete \ --access ACCESS-KEY
🔍
Discovery & Debug
Versionen & Discovery
keystone discover keystone endpoint-list --long
Bash-Integration
# Token für API-Calls: TOKEN=$(keystone token-get \ | awk '/id/ { print $4; exit }') curl -H "X-Auth-Token: $TOKEN" \ http://nova:8774/v2/servers/detail
V3 API: Für Keystone v3 mit Domains und erweiterten Rollen den openstack CLI verwenden — keystone CLI unterstützt nur v2.
