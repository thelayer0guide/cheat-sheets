Basics & Server
CLI verbinden
redis-cli redis-cli -h host -p 6379 -a password redis-cli -u redis://user:pass@host:6379 redis-cli --tls -h host
Server-Infos
PING -- Verbindung testen → PONG INFO server INFO memory DBSIZE -- Anzahl Keys CONFIG GET maxmemory CONFIG SET maxmemory 512mb FLUSHDB -- aktuelle DB leeren BGSAVE -- async Snapshot
Keys allgemein
KEYS user:* -- Pattern-Suche (prod: SCAN!) SCAN 0 MATCH user:* COUNT 100 TYPE mykey -- Typ anzeigen TTL mykey -- Restzeit in Sek EXPIRE mykey 3600 -- TTL setzen PERSIST mykey -- TTL entfernen DEL key1 key2 -- löschen EXISTS mykey

Set & Get

List-Operationen
