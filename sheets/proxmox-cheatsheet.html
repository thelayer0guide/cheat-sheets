VMs verwalten (qm)
Status & Auflistung
qm list qm status 100 pvesh get /nodes/pve/qemu # API
VM starten / stoppen / neu starten
qm start 100 qm stop 100 # sofort qm shutdown 100 # ACPI graceful qm reboot 100 qm reset 100 # hard reset qm suspend 100 qm resume 100
VM erstellen & klonen
qm create 101 \ --name myvm \ --memory 2048 \ --cores 2 \ --net0 virtio,bridge=vmbr0 \ --ostype l26 qm clone 100 200 --name myvm-clone qm clone 100 201 --full # full clone
VM konfigurieren
qm config 100 qm set 100 --memory 4096 qm set 100 --cores 4 qm set 100 --onboot 1 qm set 100 --boot order=scsi0 qm set 100 --agent 1 # QEMU agent qm set 100 --ipconfig0 ip=dhcp
Disk & CD-ROM
qm disk import 100 disk.qcow2 local-lvm qm set 100 --scsi0 local-lvm:vm-100-disk-0 qm set 100 --ide2 local:iso/debian.iso,media=cdrom qm resize 100 scsi0 +10G
Konsole & Migration
qm terminal 100 # serielle Konsole qm migrate 100 pve2 # live migration qm migrate 100 pve2 --online qm destroy 100
💡 qm guest cmd 100 network-get-interfaces — IPs per QEMU-Agent abfragen

Container verwalten (pct / LXC)
Container Status & Auflistung
pct list pct status 110 pct config 110
Container starten / stoppen
pct start 110 pct stop 110 pct shutdown 110 pct reboot 110 pct suspend 110 pct resume 110
Container erstellen
pct create 110 \ local:vztmpl/debian-12-standard_12.2-1_amd64.tar.zst \ --hostname myct \ --memory 512 \ --rootfs local-lvm:8 \ --net0 name=eth0,bridge=vmbr0,ip=dhcp \ --unprivileged 1 \ --password geheim
Container verwalten
pct enter 110 # Shell öffnen pct exec 110 -- bash -c "ls" pct set 110 --memory 1024 pct set 110 --onboot 1 pct resize 110 rootfs +5G pct clone 110 111 --hostname clone pct destroy 110
Templates verwalten
pveam update pveam available --section system pveam download local debian-12-standard_12.2-1_amd64.tar.zst pveam list local
💡 --features nesting=1 für Docker-in-LXC; --unprivileged 0 für privilegierte Container

Snapshots & Backup
VM-Snapshots
qm snapshot 100 snap1 --vmstate qm listsnapshot 100 qm rollback 100 snap1 qm delsnapshot 100 snap1 # LXC Snapshots pct snapshot 110 snap1 pct rollback 110 snap1 pct delsnapshot 110 snap1
vzdump — Backup
vzdump 100 --storage local --mode snapshot vzdump 100 --dumpdir /mnt/backup vzdump --all --storage backup # alle VMs vzdump 100 --compress zstd # Restore qmrestore /var/lib/vz/dump/vzdump-qemu-100-*.vma.zst 200 pct restore 111 /var/lib/vz/dump/vzdump-lxc-110-*.tar.zst

Netzwerk
Bridge & Bonds konfigurieren
# /etc/network/interfaces auto vmbr0 iface vmbr0 inet static address 192.168.1.10/24 gateway 192.168.1.1 bridge-ports eno1 bridge-stp off bridge-fd 0 # VLAN-Aware Bridge bridge-vlan-aware yes bridge-vids 2-4094
Netzwerk-Befehle
ifreload -a # Änderungen laden ip link show ip addr show vmbr0 brctl show pvesh get /nodes/pve/network
SDN (Software Defined Network)
pvesh get /cluster/sdn/vnets pvesh create /cluster/sdn/zones \ --zone myzone --type simple # Dann im Web-UI: Datacenter → SDN
