Images auflisten & anzeigen
Übersicht
glance image-list glance image-list --visibility public glance image-list --visibility private glance image-list --visibility shared glance image-list --status active glance image-show IMAGE-UUID
Filter
glance image-list --name ubuntu* glance image-list \ --property os_distro=ubuntu glance image-list \ --sort-key name --sort-dir asc glance image-list --limit 20 --page-size 20
⬆
Images hochladen
Erstellen & Upload (v1 API)
glance image-create \ --name ubuntu-22.04 \ --disk-format qcow2 \ --container-format bare \ --visibility public \ --file ubuntu-22.04.qcow2
Zweistufig (v2 API)
# 1. Image-Objekt anlegen: glance image-create \ --name ubuntu-22.04 \ --disk-format qcow2 \ --container-format bare \ --min-disk 8 --min-ram 512 \ --property os_type=linux \ --property os_distro=ubuntu \ --property os_version=22.04 # 2. Datei hochladen: glance image-upload \ --file ubuntu-22.04.qcow2 \ IMAGE-UUID
Disk-Formate
# qcow2 — QEMU/KVM (Standard) # raw — uncomprimiertes Rohformat # vmdk — VMware Disk # vhd — Hyper-V VHD # vhdx — Hyper-V VHDX # iso — bootbares ISO-Image # aki/ari/ami — Amazon-kompatibel
Deactivate / Reactivate
glance image-deactivate IMAGE-UUID glance image-reactivate IMAGE-UUID
⬇
Download & Löschen
Download
glance image-download \ --file ubuntu-22.04-dl.qcow2 \ IMAGE-UUID
Löschen
glance image-delete IMAGE-UUID glance image-delete UUID1 UUID2 UUID3
Update
glance image-update \ --name new-name \ --visibility shared \ IMAGE-UUID
🏷
Properties & Tags
Properties setzen
glance image-update \ --property hw_disk_bus=virtio \ --property hw_vif_model=virtio \ --property hw_scsi_model=virtio-scsi \ --property hw_qemu_guest_agent=yes \ IMAGE-UUID
Nützliche Properties
hw_disk_bus=virtio # Disk-Bus hw_vif_model=virtio # NIC-Treiber hw_qemu_guest_agent=yes # QEMU GA os_require_quiesce=yes # Quiesce f. Snaps img_config_drive=mandatory # Config-Drive hypervisor_type=qemu # Hypervisor
Tags verwalten
glance image-tag-update IMAGE-UUID ubuntu glance image-tag-update IMAGE-UUID lts glance image-tag-delete IMAGE-UUID ubuntu
🤝
Sichtbarkeit & Sharing
Sichtbarkeit ändern
# public | private | shared | community glance image-update \ --visibility public IMAGE-UUID glance image-update \ --visibility shared IMAGE-UUID
Image teilen (Member)
# Image mit Projekt teilen: glance member-create IMAGE-UUID PROJECT-UUID # Im Ziel-Projekt akzeptieren: glance member-update \ IMAGE-UUID PROJECT-UUID accepted # Übersicht: glance member-list IMAGE-UUID glance member-delete IMAGE-UUID PROJECT-UUID
community: Jeder kann ein Community-Image nutzen ohne explizite Freigabe. Gut für Team-weite Basis-Images.
