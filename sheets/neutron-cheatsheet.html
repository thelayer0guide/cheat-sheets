Netzwerke
Auflisten & Anzeigen
neutron net-list neutron net-show private neutron net-list --shared neutron net-list --router:external=True
Erstellen
neutron net-create private neutron net-create --shared shared-net neutron net-create \ --router:external \ --provider:network_type flat \ --provider:physical_network physnet1 \ external
Provider-Netzwerke
neutron net-create \ --provider:network_type vlan \ --provider:physical_network physnet1 \ --provider:segmentation_id 100 \ vlan100 neutron net-create \ --provider:network_type vxlan \ --provider:segmentation_id 200 \ vxlan200 neutron net-update private --name internal neutron net-delete private
📡
Subnetze
Erstellen IPv4
neutron subnet-create private 10.0.0.0/24 \ --name mysubnet \ --gateway 10.0.0.1 \ --dns-nameserver 8.8.8.8 \ --dns-nameserver 8.8.4.4 \ --allocation-pool \ start=10.0.0.100,end=10.0.0.200
Erstellen IPv6
neutron subnet-create private fd00::/64 \ --name mysubnet-v6 \ --ip-version 6 \ --ipv6-ra-mode slaac \ --ipv6-address-mode slaac
Verwalten
neutron subnet-list neutron subnet-show mysubnet neutron subnet-update mysubnet --no-dhcp neutron subnet-update mysubnet --enable-dhcp neutron subnet-delete mysubnet
🔀
Router
Erstellen & Gateway
neutron router-create myrouter neutron router-create --ha myrouter # HA via VRRP neutron router-gateway-set myrouter external neutron router-gateway-clear myrouter
Interfaces
neutron router-interface-add myrouter mysubnet neutron router-interface-delete myrouter mysubnet neutron router-port-list myrouter
Statische Routen
neutron router-update myrouter \ --routes type=dict,list=true \ destination=192.168.0.0/24,nexthop=10.0.0.254
Verwalten
neutron router-list neutron router-show myrouter neutron router-delete myrouter
HA Router: --ha erzeugt einen Router mit VRRP-Failover. Erfordert Neutron L3HA-Konfiguration.
🔌
Ports
Erstellen & Konfigurieren
neutron port-create private --name myport neutron port-create private \ --fixed-ip ip_address=10.0.0.50 \ --mac-address aa:bb:cc:dd:ee:ff \ --name myport neutron port-update myport \ --security-group websg
Allowed Address Pairs
neutron port-update myport \ --allowed-address-pairs \ type=dict,list=true \ ip_address=10.0.0.0/24 neutron port-update myport \ --no-security-groups # alle SGs entfernen
Verwalten
neutron port-list neutron port-list --network private neutron port-show myport neutron port-delete myport

Floating IPs
Erstellen & Zuweisen
neutron floatingip-create external neutron floatingip-associate FIP-UUID PORT-UUID neutron floatingip-disassociate FIP-UUID
Verwalten
neutron floatingip-list neutron floatingip-show FIP-UUID neutron floatingip-delete FIP-UUID

Security Groups
Erstellen
neutron security-group-list neutron security-group-create websg \ --description "Web Server SG" neutron security-group-delete websg
Regeln hinzufügen
# SSH eingehend: neutron security-group-rule-create \ --protocol tcp \ --port-range-min 22 --port-range-max 22 \ --remote-ip-prefix 10.0.0.0/8 websg # HTTP/HTTPS: neutron security-group-rule-create \ --protocol tcp \ --port-range-min 80 --port-range-max 80 websg neutron security-group-rule-create \ --protocol tcp \ --port-range-min 443 --port-range-max 443 websg # ICMP (ping): neutron security-group-rule-create \ --protocol icmp websg # Egress alles: neutron security-group-rule-create \ --direction egress websg
Regel löschen
neutron security-group-rule-list websg neutron security-group-rule-delete RULE-UUID
