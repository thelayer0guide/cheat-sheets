Suchen & Ersetzen
Suchen-Toolbar (Find Bar)
Ctrl+F # Suchleiste unten F3 / Shift+F3 # nächster / vorheriger Treffer Esc # Suchleiste schließen
Suchen & Ersetzen-Dialog
Ctrl+H
Reguläre Ausdrücke aktivieren
# im Dialog Suchen & Ersetzen: ☑ Reguläre Ausdrücke \b\w+\b # Beispiel: ganze Wörter \n # Zeilenumbruch \t # Tabulator

Ctrl+Shift+F # Find & Replace (vollständig) # Optionen: Groß-/Kleinschreibung, # Ganzes Wort, Wildcards

Tabellen

Ctrl+F12 # Tabelle einfügen-Dialog # oder Einfügen → Tabelle
Navigation in Tabellen
Tab # nächste Zelle Shift+Tab # vorherige Zelle Tab (letzte Zelle) # neue Zeile anfügen
Zeile / Spalte einfügen
# Rechtsklick in Tabelle → Zeilen einfügen (oben/unten) Spalten einfügen (links/rechts) # oder Tabelle-Menü

Felder & Serienbriefe
Feld einfügen (Datum, Seitenzahl …)
Einfügen → Feld → Datum (fixiert) # statisches Datum Datum (variabel) # aktuell beim Drucken Seitennummer # aktuelle Seite Seitenanzahl # Gesamtzahl Dateiname

Extras → Serienbriefe # Schritt 1: Dokumenttyp (Brief / E-Mail) # Schritt 2: Datenquelle wählen # (CSV, Calc-Tabelle, Datenbank) # Schritt 3: Felder einfügen # Schritt 4: Alle zusammenführen

Einfügen → Feld → Weitere Felder # Kategorie: Datenbank # Typ: Serienbrieffeld

Ctrl+Alt+C # Kommentar einfügen # oder Einfügen → Kommentar
Kommentare anzeigen / ausblenden
Ansicht → Kommentare anzeigen
Änderungsverfolgung ein/aus

Export
Als PDF exportieren
Datei → Exportieren als → PDF-Dokument … # kein Standard-Tastenkürzel # Optionen: Bereich, Komprimierung, # Lesezeichen, Formulare, Wasserzeichen
Direkt als PDF drucken
Datei → Exportieren als PDF … # ☑ Automatisch (kein Dialog) # oder CLI: soffice --headless # --convert-to pdf dokument.odt
Als DOCX speichern
Datei → Speichern unter # Dateityp: Word 2007–365 (.docx) Ctrl+Shift+S
Als ODT speichern (Standard)
Ctrl+S # speichert im Standardformat # Standardformat: ODT (ODF Text)
Kommandozeilen-Konvertierung
soffice --headless \ --convert-to docx dokument.odt soffice --headless \ --convert-to pdf *.odt
Kompatibilität: Datei → Optionen → Laden/Speichern → Allgemein: Standardformat auf DOCX umstellen für Microsoft-Workflows.

Makros & Basic IDE
Makro aufzeichnen
Extras → Makros → Makro aufzeichnen # Aktionen ausführen → Aufzeichnung stoppen
Makro ausführen
Extras → Makros → Makro ausführen… Alt+F8 # Makro-Dialog öffnen
Basic IDE öffnen
Extras → Makros → Basic-IDE bearbeiten Alt+F11 # direkt in die IDE
Einfaches Writer-Makro (Basic)
Sub HalloWriter() Dim oDoc As Object Dim oText As Object oDoc = ThisComponent oText = oDoc.getText() Dim oCursor As Object oCursor = oText.createTextCursor() oText.insertString(oCursor, _ "Hallo Writer!", False) End Sub
Makro an Ereignis binden
Extras → Anpassen → Ereignisse # Ereignis wählen → Makro zuweisen # z.B. Dokument speichern → Sub OnSave()
Sicherheit: Extras → Optionen → LibreOffice → Sicherheit → Makrosicherheit auf "Mittel" für eigene Makros.
