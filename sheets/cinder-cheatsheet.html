Volumes
Auflisten & Anzeigen
cinder list cinder list --all-tenants cinder list --status error cinder show myvol
Erstellen
cinder create --display-name myvol 100 # Aus Image (bootbar): cinder create \ --image-id IMAGE-UUID \ --volume-type ssd \ --display-name bootvol 30 # Aus Snapshot: cinder create \ --snapshot-id SNAP-UUID \ --display-name vol-from-snap 100 # Aus bestehendem Volume (Clone): cinder create \ --source-volid VOL-UUID \ --display-name vol-clone 100
Ändern & Löschen
cinder rename myvol newname cinder extend myvol 200 # nur vergrößern! cinder set-bootable myvol true cinder metadata myvol set key=value cinder delete myvol cinder force-delete myvol
🔗
Attach & Detach
Attach an Server
cinder attach myvol SERVER-UUID /dev/vdb
Detach
cinder detach myvol cinder detach --attachment-uuid ATT-UUID myvol
Attachment-API (v3)
cinder attachment-list cinder attachment-create VOL-UUID SERVER-UUID cinder attachment-complete ATT-UUID cinder attachment-delete ATT-UUID
Multiattach: cinder create --allow-multiattach --display-name shared-vol SIZE — erfordert einen Volume-Typ mit multiattach=true.
📸
Snapshots
Erstellen & Verwalten
cinder snapshot-list cinder snapshot-create --display-name mysnap myvol cinder snapshot-create --force --display-name mysnap myvol cinder snapshot-show mysnap cinder snapshot-rename mysnap newname cinder snapshot-delete mysnap
Aus Snapshot erstellen
cinder create \ --snapshot-id SNAP-UUID \ --display-name vol-from-snap 100
🔒
Backups
Erstellen
cinder backup-create --name mybak myvol cinder backup-create \ --name mybak-inc \ --incremental myvol
Restore & Verwalten
cinder backup-list cinder backup-show mybak cinder backup-restore mybak cinder backup-restore \ --volume-id VOL-UUID mybak cinder backup-delete mybak
🏷
Volume Types & QoS
Volume Types
cinder type-list cinder type-show ssd cinder type-create ssd cinder type-key ssd set \ volume_backend_name=ceph-ssd cinder type-key ssd set \ multiattach=" True" cinder type-delete ssd
Encryption
cinder encryption-type-create \ --cipher aes-xts-plain64 \ --key-size 256 \ --control-location front-end \ encrypted \ nova.volume.encryptors.cryptsetup.CryptsetupEncryptor
QoS Specs
cinder qos-create 5k-iops \ consumer=back-end \ total_iops_sec=5000 cinder qos-associate 5k-iops TYPE-UUID cinder qos-list cinder qos-show 5k-iops
🔄
Transfers & Quotas
Volume Transfer
# Volume an anderes Projekt übergeben: cinder transfer-create myvol # Gibt transfer-id + auth-key aus # Im Ziel-Projekt annehmen: cinder transfer-accept \ TRANSFER-UUID AUTH-KEY
Quotas
cinder quota-show PROJECT-UUID cinder quota-update \ --volumes 100 \ --gigabytes 10000 \ --snapshots 200 \ PROJECT-UUID
Admin: Manage / Unmanage
# Externes Volume importieren: cinder manage \ --name myvol \ cinder@lvm#LVM \ VOLUME-REF # Cinder-Verwaltung entfernen: cinder unmanage myvol
