VM Prozesse
VM-Liste & Kill
esxcli vm process list esxcli vm process list | grep -i myvm esxcli vm process kill --type soft --world-id 12345 esxcli vm process kill --type hard --world-id 12345 esxcli vm process kill --type force --world-id 12345
Reihenfolge: soft → hard → force. Immer mit soft starten — force sollte nur als letztes Mittel verwendet werden.
